# ticTacToe
Tic Tac Toe Game

## Technologies used

Made with:
- Webpack
- Node
- Yarn
- React
- Redux
- Typescript
- CSS Modules

# Getting Started

## Installation

If you are using nvm, you can switch to the supported npm version with:

```bash
nvm use
```

Install node packages with:

```bash
yarn
```

Finally you can start the dev server with:

```bash
yarn dev
```

And navigate to http://localhost:8080 to see it running
